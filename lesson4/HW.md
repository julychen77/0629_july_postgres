##7.台灣有多少人在2020確診

```sql
SELECT MAX(總確診數) AS 台灣2020年總確診數
FROM world
WHERE 國家 ='台灣' AND 日期 BETWEEN '2020-01-01' AND '2020-12-31' ;
```
| 台灣2020年總確診數 | 備註 |
| --- | --- |
| 799 |答案 |

##查詢亞洲總共有多少人死亡
```sql
SELECT MAX(總死亡數) AS 亞洲總死亡數
FROM world
WHERE 洲名 ='亞洲';
```
| 亞洲總死亡數 | 備註 |
| --- | --- |
| 52,305 |答案 |
